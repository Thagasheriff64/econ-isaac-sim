# econ.itof.menu

Adds **e-con DepthVista Helix iToF** cameras to
**Create → Sensors → Camera and Depth Sensors → e-con** in Isaac Sim:

- DepthVista Helix iToF (USB)
- DepthVista Helix iToF (GMSL)

Clicking an entry references the matching USD (`assets/DEPTH_VISTA_HELIX_*.usd`)
into the current stage. Pure-Python — nothing to compile.
